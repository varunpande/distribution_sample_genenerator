About

{% if dist_type == "bernoulli" %}

Used to model experiments with a binary outcome (yes or no,pass or fail,true or false)

{% elif dist_type == "binomial" %}

It is used to model number of success in a sequence of independent Bernoulli trials. Models the probability of x successes in n trials.

{% elif dist_type == "geometric" %}

It is used to model number of trials needed to achieve the first success in a sequence of independent Bernoulli trials. Models the probabilty of xth successive trial resulting in a success.

{% elif dist_type == "neg-binomial" %}

It is used to model number of trials needed to obtain k success in a sequence of independent Bernoulli trials. Models the probability of xth successive trial resulting in the kth success.

{% elif dist_type == "poisson" %}

It is used to model number of rare events occurring within a fixed period of time. Models the probability of x rare events occurring in a fixed period of time if we know the frequency at which the events occur on average.

{% elif dist_type == "arb-discrete" %}

samples for any arbitrary discrete distribution.

{% elif dist_type == "exponential" %}

Used to model time (or seperation) between events occuring at frequency lambda(rate at which the events occur).

{% elif dist_type == "gamma" %}

Used to model total time of multistage processes with alpha steps (shape parameter) where time of each step can be modeled as an Exponential distribution with frequency lambda.

{% elif dist_type == "normal" %}

Used to model a large number of scenarios, due to central limit theorem(C.L.T) a large number of samples of any distribution tend to behave like normal distribution.

{% endif %}
